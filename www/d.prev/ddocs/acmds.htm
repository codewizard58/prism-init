Builtin commands

All builtin commands start with @. The uppercase characters show the minimum letters that need to be typed for the command to be matched.

    @ARGs ................ ........... Print the argument table
    @BOOLfuncs page_no ... ........... List lock functions
    @BRoadcast msg ....... ... Broadcast 'msg'
    @CHown obj=new_owner . ... Change Ownership of obj
    @CLass obj=class_room. ... Change Ownership of obj
    @COMmands page_no ................ List this info
    @COST obj=new_value .. ... Change the pennies value of an object
    @CReate obj=value ....  Create a Thing Object with name 'obj'
    @DEBUG number ........ .... Used to set debug flag
    @DEFault type=class .. .... Used to set default classes
    @DESCribe obj=desc ...  Set the desciption of obj to 'desc'
    @DESTROY obj .........  Destroy obj, you must own it.
    @DIG name [=exit] ....  Create a new room
    @DUMP ................ .... Does a checkpoint dump
    @EDIT obj ............ ... Edit object
    @EXamine obj .........  Get detailed look of object you own
    @EXTRACT arg1=arg2 ...  ..... Extract part of the database
    @FIND [/exit|/#obj] name ......... Find objects with 'name' in their title
    @FLAgs ............... ........... Print the flag values
    @Gripe msg ....................... Complain
    @HELp ............................ Print the help file
    @Inventory ...........  ..... See your contents
    @LBRoadcast msg ......  ..... Broadcast 'msg' within ZONE
    @LEVels .......................... Print the levels info
    @LInk obj = loc ......  . Link exit 'obj' to room 'loc'
    @LOCK obj = expr .....  . Lock the obj with the lock 'expr'
    @LOOK obj ............  ..... Look at obj
    @MERGE file ..........  ..... Upload another TinyDatabase
    @NAME old = new .................. Change the name of obj 'old to 'new'
    @NEWPASSWORD player=password. 
    @NEWs ............................ Print the news file
    @OPEN [/#obj] exit[= #room] ..  . Open exit [command on obj] to room
    @PASSWORD old_pass=new_pass ...... Change your password
    @QUIT ............................ Leave game.
    @RELInk exit=#room ...  . Relink an exit/command to room
    @RESET ...............  ..... Perform a reset, send .reset # to all
    @ROBOt name=stats ....  ..... Control robots
    @SCore ........................... Tell you your score
    @SET obj = [!]flag ...  . Set/Reset flags on obj
    @SHUTDOWN ............  ..... Save the database and shutdown
    @STATS	.......................... Print various statistics
    @TIME obj[=delta] ....  ..... Change the timestamp of object
    @TOAD player .........  .... Turn a player into toad (Non reversable)
    @TOP ................. ........... Print the top ten.
    @UNLINK obj ..........  . Unlink an exit
    @UNLOCK obj ..........  . Remove the lock on object
    @VERsion ............. ........... Print version number
    @WHICH command .......  . Print the exits that match command
    @ZONE ................ ........... Print the zone you are in
