Object flags

The bottom 4 bits of the flag is used to determine the type of the object. Other flags are used for various controls.

    16 Pseudo Flag used to check if object is logged in.

In PRISM the main flags that provide a lot of the new functionality are

INHERIT
When set on a room, this indicates that the room is a CLASS object (classroom ) and other objects can specify this room as their class. Another use of the INHERIT flag on a room designates the room as a Zone object.
FEMALE
The F flag set on a player indicates that they are 'female'. When set on an exit/command, this indicates that the search should stop here.
